# Sheila On 7 – Dan...

> **歌詞登録ステータス：プレーン歌詞のみ**

```
Dan...
Bila esok datang kembali
Seperti sedia kala dimana kau
bisa bercanda
Dan...
Perlahan kaupun, lupakan aku
Mimpi burukmu
Dimana t'lah kutancapkan duri
tajam
Kaupun menangis, menangis sedih
Maafkan aku...
Dan...
Bukan maksudku, bukan inginku
Melukaimu sadarkah kau di sini
'kupun terluka
Melupakanmu, menepikanmu
Maafkan aku
Lupakanlah/caci maki saja diriku
Bila itu bisa membuatmu kembali
bersinar
Dan berpijar seperti dulu kala
```
